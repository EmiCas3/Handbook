\markdownRendererHeadingOne{Algoritmo de Kadane}\markdownRendererInterblockSeparator
{}El algoritmo de Kadane resuelve el problema del subarreglo de suma máxima en tiempo lineal O(n) mediante programación dinámica. Mantiene dos variables: la suma máxima terminando en la posición actual y la suma máxima global encontrada. Itera el arreglo una vez, decidiendo en cada paso si extender el subarreglo actual o iniciar uno nuevo, demostrando que la solución óptima puede construirse incrementalmente sin explorar todas las O(n²) posibilidades.\markdownRendererParagraphSeparator
