{}El algoritmo de Manacher encuentra todos los palíndromos en una cadena en tiempo lineal $O(n)$. Utiliza simetría para evitar recalcular información ya conocida, manteniendo el centro y radio del palíndromo más a la derecha encontrado. Transforma la cadena insertando caracteres especiales para manejar uniformemente palíndromos de longitud par e impar, siendo significativamente más eficiente que el enfoque $O(n²)$ de expansión desde el centro.\markdownRendererInterblockSeparator
