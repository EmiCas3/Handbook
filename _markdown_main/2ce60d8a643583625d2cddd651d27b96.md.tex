\markdownRendererHeadingOne{Algoritmo de Tarjan para Componentes Fuertemente Conexos}\markdownRendererInterblockSeparator
{}El algoritmo de Tarjan identifica componentes fuertemente conexas en grafos dirigidos en $O(V+E)$ mediante DFS con un solo recorrido. Utiliza una pila y dos arreglos: tiempos de descubrimiento y valores "low" que rastrean el ancestro más bajo alcanzable. Cuando un nodo tiene $low[v] = disc[v]$, identifica la raíz de una componente fuertemente conexa. Este algoritmo es fundamental para análisis de conectividad, detección de ciclos y problemas de satisfacibilidad.\markdownRendererInterblockSeparator
