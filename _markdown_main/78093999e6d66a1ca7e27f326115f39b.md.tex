\markdownRendererHeadingOne{Knuth-Morris-Pratt KMP}\markdownRendererInterblockSeparator
{}El algoritmo KMP busca patrones en texto con complejidad $O(n+m)$ mediante un preprocesamiento que evita comparaciones redundantes. Construye un arreglo de prefijos que almacena para cada posición la longitud del mayor prefijo propio que es también sufijo. Durante la búsqueda, cuando ocurre un desajuste, el algoritmo utiliza esta información para desplazar el patrón inteligentemente sin retroceder en el texto, eliminando el comportamiento $O(nm)$ del enfoque ingenuo.\markdownRendererParagraphSeparator
